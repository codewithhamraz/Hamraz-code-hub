Chapter 1: Modules, Comments, and PIP Frameworks

Python ek high-level, human-readable language hai jo clean design logic par kaam karti hai. Is module mein hum computational syntax ki shuruat karenge.

Module: Module ek aisi file hoti hai jisme pehle se likha hua code hota hai, jise hum apne project mein scale badhane ke liye use karte hain. Python mein do tarah ke modules hote hain: Built-in (jo pehle se aate hain) aur External (jo baahar se install karne padte hain).

💡 PIP: PIP Python ka official package manager hai jiske bina aap external libraries install nahi kar sakte.

pip install flask requests numpy
